## Belangrijkste Functionaliteiten

<AccordionGroup>
  <Accordion title="AI-gestuurde Documentverwerking">
    Automatische extractie van factuurgegevens met behulp van geavanceerde AI-modellen en OCR-fallback voor gescande documenten.
  </Accordion>
  
  <Accordion title="Intelligente 3-Weg Matching">
    Automatische matching van facturen met inkooporders en ontvangstbewijzen voor naadloze verificatie.
  </Accordion>
  
  <Accordion title="E-mail Integratie">
    Directe integratie met Microsoft Outlook voor automatische verwerking van facturen uit e-mailbijlagen.
  </Accordion>
  
  <Accordion title="LangGraph Workflow Orchestratie">
    Krachtige agent-gebaseerde architectuur voor complexe documentverwerkingsworkflows.
  </Accordion>
</AccordionGroup>

## Technologie Stack

<Tabs>
  <Tab title="Backend">
    - **Framework**: FastAPI (Python 3.13+)
    - **AI/ML**: LangChain 0.3.22+, LangGraph 0.3.21+
    - **Database**: Supabase (PostgreSQL Cloud)
    - **Cache**: Redis
    - **Monitoring**: Langfuse, Prometheus, Grafana
  </Tab>
  
  <Tab title="Frontend">
    - **Framework**: Next.js 14 met TypeScript
    - **AI Integratie**: CopilotKit 1.9.3+
    - **UI Componenten**: Radix UI, Tailwind CSS
    - **Package Manager**: Bun
  </Tab>
  
  <Tab title="Integraties">
    - **E-mail**: Microsoft Outlook API
    - **Boekhouding**: AccountView Export
    - **Opslag**: Supabase Storage
    - **Authenticatie**: Supabase Auth met JWT
  </Tab>
</Tabs>

## Aan de Slag

```bash
# Clone de repository
git clone https://github.com/sky-dust-intelligence/3wm-backend.git

# Installeer met Docker Compose
docker-compose up -d

# Of installeer handmatig
uv venv
source .venv/bin/activate  # Op Windows: .venv\Scripts\activate
uv pip install -e .
```

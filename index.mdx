## Core Features

- **Advanced OCR**: State-of-the-art text extraction with DocTR and TrOCR
- **AI Agents**: LangGraph-powered agents for intelligent document analysis
- **Knowledge Graph**: Neo4j integration for relationship mapping
- **Vector Search**: Semantic search and duplicate detection
- **Multi-Database**: Optimized data storage across PostgreSQL, Neo4j, and Redis
- **Enterprise Ready**: JWT auth, rate limiting, monitoring, and scalability

## Quick Example

```bash
# Upload an invoice
curl -X POST https://api.skydust.app/api/v1/upload \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@invoice.pdf"

# Query with natural language
curl -X POST https://api.skydust.app/api/v1/agent/query \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"query": "What is the total amount of invoices from ACME Corp this month?"}'
```

## Architecture

3WM uses a modern microservices architecture designed for scalability:

- FastAPI backend for high-performance REST APIs
- LangChain/LangGraph for AI agent orchestration
- Docker containerization for easy deployment
- Kubernetes ready for enterprise scale

## Intelligente Data Extractie

### Context-Aware Field Extraction

```python
# Voorbeeld van intelligente extractie met context
async def extract_invoice_data(text: str, image: bytes):
    # Multi-modal analyse
    visual_context = await analyze_layout(image)
    
    # Prompt engineering voor Nederlandse context
    prompt = f"""
    Analyseer deze Nederlandse factuur en extraheer:
    - Leverancier details (inclusief KVK/BTW nummer)
    - Factuurgegevens (nummer, datum, vervaldatum)
    - Regelitems met Nederlandse BTW tarieven (21%, 9%, 0%)
    - Totaalbedragen (excl. BTW, BTW bedrag, incl. BTW)
    
    Context: {visual_context}
    Tekst: {text}
    
    Retourneer als JSON met Nederlandse veldnamen.
    """
    
    response = await llm.invoke(prompt, response_format="json")
    return validate_and_enrich(response)
```

### Confidence Scoring

<Note>
  Elk geëxtraheerd veld krijgt een confidence score, waardoor gebruikers weten wanneer menselijke verificatie nodig is.
</Note>

#### Confidence Levels

| Score | Betekenis | Actie |
|-------|-----------|-------|
| > 0.95 | Zeer zeker | Automatisch goedkeuren |
| 0.80-0.95 | Waarschijnlijk correct | Flaggen voor spot check |
| 0.60-0.80 | Onzeker | Menselijke review vereist |
| < 0.60 | Onbetrouwbaar | Handmatige invoer nodig |

## Overzicht

Het 3WM systeem is een geavanceerd AI Document Intelligence platform dat speciaal ontwikkeld is voor de Brouwer Group. Het systeem automatiseert het complete factuurverwerkingsproces van email ontvangst tot boeking in AccountView.

<Info>
Van 10 minuten handwerk naar 10 seconden AI-verwerking per factuur met 99.9% nauwkeurigheid.
</Info>

## Het Complete Verwerkingsproces

<Steps>
  <Step title="Exchange Online">
    Facturen komen binnen op **administratie@brouwer-group.nl**
  </Step>
  <Step title="AI Processing">
    6 gespecialiseerde AI assistenten verwerken het document
  </Step>
  <Step title="AccountView">
    Automatische boeking met confidence-based routing
  </Step>
</Steps>

### 2. Intelligente Data Extractor
**NLP Engine met Machine Learning**

Begrijpt de context van facturen en haalt automatisch alle relevante gegevens eruit.

```
INVOICE_EXTRACTOR {
  - Leverancier Herkenning (Named Entity Recognition)
  - Factuurnummer Detectie (Pattern Matching)
  - Datum Parsing (Multiple Format Support)
  - Bedrag Extractie (Currency Handling)
  - BTW Analyse (Rate Detection + Validation)
  - IBAN/KVK Verificatie (Checksum Validation)
  - Factuurregels (Table Extraction)
}
```

### 3. Semantische Zoek AI

### 4. Boekhoud Predictor AI
**ML Journal Engine voor AccountView**

Voorspelt automatisch de correcte journaalposten voor AccountView op basis van historische patronen.

<Note>
**Beslisregels:**
- **95%+ confidence:** Automatisch boeken (~85% van facturen)
- **80-94% confidence:** Ter controle aanbieden (~12% van facturen)
- **&lt;80% confidence:** Handmatige invoer vereist (~3% van facturen)
</Note>

## Technische Architectuur

- **Backend:** FastAPI met JWT authenticatie
- **Databases:** PostgreSQL (auth), Supabase (business data), Neo4j (graph), Redis (cache)
- **AI Models:** GPT-4 Turbo, text-embedding-3-small
- **OCR:** DocTR engine
- **Integraties:** Microsoft Graph API, AccountView API
